# PetCare+ | Módulo de Autenticación

Este proyecto es el prototipo del módulo de **Login** para la plataforma **PetCare+**, una aplicación diseñada para la gestión y cuidado de mascotas.

El objetivo principal fue desarrollar una interfaz de usuario (UI) limpia, moderna y responsiva, implementando lógica de validación en el lado del cliente (Client-side validation).

**[Ver Demo en Vivo aquí](https://juanj983009-eng.github.io/PetCare-Frontend/)**

---

##  Tecnologías Utilizadas
* **HTML5 Semántico:** Estructura optimizada y accesible.
* **CSS3:** Estilos personalizados, uso de **Flexbox** para alineación y diseño de tarjetas (Card Design) con sombras y bordes redondeados.
* **JavaScript (ES6):** Manipulación del DOM para capturar eventos y validar credenciales básicas.
* **Git & GitHub:** Control de versiones y despliegue continuo con GitHub Pages.

##  Funcionalidades
1.  **Interfaz Moderna:** Diseño minimalista centrado en la experiencia de usuario (UX).
2.  **Validación de Inputs:** El sistema detecta si los campos están vacíos antes de enviar datos, proporcionando feedback visual inmediato mediante alertas.
3.  **Interactividad:** Estados *hover* en botones y feedback visual al interactuar.
